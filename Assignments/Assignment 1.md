# Problem 1: Basic
Calculate the factorial of a number
Input: A positive integer N
Output: N! (N factorial)

### Algorithm:
1. START.
2. Input N.
3. Set Fact = 1
4. Repeat for i from 1 to N:
    fact = fact * i
5. After loop ends, fact will hold the required factorial of N.
6. Output fact
7. End

### Psuedocode:

    BEGIN Factorial
        Input N
        Set fact to 1
        For i = 1, if i <= N, Do 
              fact = fact * i
              i = i + 1
        End For
        Output "Factorial is" : Fact
    End Factorial
